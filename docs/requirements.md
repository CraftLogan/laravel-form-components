## General Requirements

-   PHP **8.0** or greater
-   Laravel **8.0** or greater
-   TailwindUI for styling
-   Alpine.js
-   Popper.js (when using components like custom-select)
-   Blade Heroicons (can easily swap for icon libraries)

| ------- | --------------- |
| 8.0     | 1.0.0           |
